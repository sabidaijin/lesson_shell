# シェルスクリプトで学ぶ「検索・抽出・ログ解析」入門

## このリポジトリについて

初心者向けに、grep・awk・sed・jq などの基本コマンドを「実際に動かしながら学べる」練習用スクリプトです。
このスクリプトは、Linux の基本操作＋テキスト処理コマンドの体験教材です。
1つのファイル (lesson.sh) に、学習用の「章」ごとに関数がまとめられています。
※以下の環境でのみ動作確認済み

- Bash 5.2.21  
- jq 1.7  
- Ubuntu 22.04（WSL2環境）  
- CPU: x86_64 (Linux カーネル)

| 章        | 内容     | 学べるコマンド           |
| -------- | ------ | ----------------- |
| grep     | 検索     | ログからキーワードを抜き出す    |
| awk      | 整形     | 行・列を指定してデータを扱う    |
| cut      | 抽出     | 区切り文字でカラムを切り出す    |
| sed      | 編集     | テキストを置換・削除        |
| jq       | JSON処理 | JSONからデータ抽出       |
| sort     | 並べ替え   | 行を昇順・降順にソート       |
| uniq     | 重複処理   | 重複行を削除・カウント       |
| wc       | 集計     | 行数・単語数・文字数のカウント   |
| regex    | 正規表現   | パターン検索の基本         |
| redirect | 出力制御   | 標準出力／エラー出力のリダイレクト |
